# Тестовые задания

Репозиторий содержит два тестовых задания.

1. Написать скрипт, который читает список пользователей из CSV и выводит в консоль отфильтрованный результат.
2. Написать скрипт, который забирает случайный факт о котах из публичного API и сохраняет его в файл.

## Стек

- Python 3.13
- Poetry
- httpx
- pydantic
- pytest

## Установка зависимостей

Перед запуском установить зависимости:

```bash
poetry install
```

## Запуск

### Task 1 — Консольный справочник

Запуск из корня проекта:

```bash
poetry run python -m task_1.script "<путь_к_csv>"
```


### Task 2 — Сборщик фактов

Запуск из корня проекта:

```bash
poetry run python -m task_2.main
```

## Тесты

Тесты второго задания:

```bash
poetry run pytest task_2/tests
```
